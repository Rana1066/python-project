2. The user tries to guess the number.
3. The random module is used to create the number.
4. The user enters a guess using input().
5. The input is converted to an integer.
6. The program compares the guess with the secret number.
7. If the guess is smaller, it shows "Too low".
8. If the guess is greater, it shows "Too high".
9. If the guess is correct, the user wins.
10. A while loop keeps the game running.
11. The loop stops when the correct answer is found.
12. Conditional statements control the game logic.
13. Variables store the numbers.
14. The project improves problem-solving skills.
15. This project teaches loops, conditions, and user input.
